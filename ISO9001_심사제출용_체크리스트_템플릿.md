# ISO 9001 심사 제출용 체크리스트 템플릿

## 1) 사용 방법
- 심사 기준일: `YYYY-MM-DD`
- 적용 범위: `예) 자동차 부품 가공/검사`
- 작성 책임자: `이름/부서`
- 판정 기준:
  - `적합`: 요구 증빙이 최신 상태로 확인됨
  - `부분적합`: 증빙 일부 존재하나 누락/미완료 있음
  - `부적합`: 증빙 없음 또는 요구 불충족

## 6) 경영검토 회의록 템플릿 (9.3)
### 기본정보
- 회의일: `YYYY-MM-DD`
- 참석자: `대표/품질/생산/구매/영업`
- 대상기간: `예) 2026-01 ~ 2026-03`

### 입력사항
- 이전 경영검토 후속조치 상태
- 품질목표 달성 현황
- 고객불만/클레임 추이

- 공급자 성과
- 자원/교육/설비/계측기 이슈
- 리스크 및 기회 변경사항

## 10) 심사 직전 최종 점검(체크박스)
- [ ] 조항별 체크리스트 상태가 모두 업데이트되었는가
- [ ] `부분적합/부적합` 항목에 시정조치 계획과 기한이 있는가
- [ ] CAPA 종료건의 효과검증 근거(수치/그래프)가 있는가
- [ ] 문서 개정본과 현장 사용본 Rev가 일치하는가
- [ ] 내부심사 지적사항 미해결 건의 리스크가 설명 가능한가
- [ ] 경영검토 결정사항이 책임자/기한으로 관리되는가
- [ ] 대시보드 수치와 원본 데이터(엑셀/DB)가 일치하는가
