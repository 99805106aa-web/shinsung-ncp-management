# PRD_고객사별_월별현황_인쇄개선_분석강화_v1

- 작성일: 2026-03-04
- 대상 화면: `tab-customer-monthly` (고객사별 월별현황)
- 버전: v1.0

## 1) 배경
현재 고객사별 월별현황 탭은 화면 조회 중심으로 구성되어 있어, 월별 게시용 인쇄물로 출력 시
- 정보 밀도/가독성(표, KPI, 차트 배치)
- 인쇄 시 불필요 UI 노출(필터/버튼 등)
- 월별 분석 근거(추세/이상징후)의 정량 정보 부족
문제가 있다.

## 2) 목표
1. 월 단위 게시용 인쇄 결과물을 한 번에 출력할 수 있게 한다.
2. 인쇄물에서 핵심 숫자(KPI), 월별 집계, 유형 분포, 추세/이상징후를 한눈에 확인 가능하게 한다.
3. 월별 분석기법(이동평균 + 관리한계)을 시각화하여 회의/게시용 설명력을 높인다.

## 3) 사용자 시나리오
- 품질 담당자가 고객사/연도/월/기준(수량·건수)을 선택한다.
- "월별현황 인쇄" 버튼을 누른다.
- 인쇄물에는 해당 선택 조건이 헤더에 명시되고, 게시 가능한 형태로 출력된다.
- 분석 요약에서 전월 대비 변화와 이상징후 여부를 즉시 확인한다.

## 4) 범위
### 포함
- 고객사별 월별현황 탭 전용 인쇄 모드
- 월 선택 필터 추가
- 월별 분석 요약 카드/문장 추가
- 불량율 관리도(중심선/관리상한·하한/3개월 이동평균) 추가

### 제외
- PDF 자동 저장/배포 자동화
- 백엔드/서버 API 변경
- 기존 다른 탭(사내/사외/월별보고) 구조 개편

## 5) 기능 요구사항
### FR-1. 월 필터
- 연도/고객사 조건에 맞는 월 옵션을 제공한다.
- 월=전체/특정월 선택 가능.
- KPI/표/원형차트 집계는 선택 월을 반영한다.

### FR-2. 인쇄 전용 모드
- 버튼: `월별현황 인쇄`
- 인쇄 시 `tab-customer-monthly`만 출력.
- 화면용 컨트롤(필터 버튼군/토글 등)은 숨김.
- A4 가로(landscape) 기준 게시용 1페이지 레이아웃으로 최적화.
- 인쇄 헤더에 다음 정보 표시:
  - 고객사
  - 연도/월
  - 기준(수량/건수)
  - 출력 일시

### FR-3. 월별 분석 요약
- 분석 카드 4종 제공:
  1. 기준월 불량율(또는 건수 불량율)
  2. 전월 대비 증감(절대값 + %)
  3. 3개월 이동평균
  4. 관리한계(UCL/LCL) 대비 상태(안정/주의/이상)
- 텍스트 인사이트 3~4줄 제공:
  - 추세 해석
  - 변동성 해석
  - 상위 불량유형 집중도
  - 조치 우선순위

### FR-4. 불량율 관리도
- X축: 월
- Y축: 불량율(%)
- 선/점: 실제 불량율
- 보조선: 3개월 이동평균(MA3)
- 기준선: 중심선(CL=평균), UCL/LCL(평균 ± 3σ)
- 관리한계 이탈점 강조 마커

## 6) 비기능 요구사항
- 기존 탭 로딩/전환 동작을 해치지 않아야 함.
- 데이터 없음/1개월 데이터 등 경계조건에서 오류 없이 안내 문구를 표시.
- 인쇄 시 표 텍스트 크기/간격이 A4 기준 가독성을 만족해야 함.

## 7) UI/UX 설계 요약
1. 필터 영역에 `월 선택` 추가.
2. KPI 하단에 `월별 분석 요약`(카드+문장) 섹션 추가.
3. `불량율 관리도` 차트 섹션 추가.
4. 인쇄 시 단일 컬럼 흐름으로 재배치(박스 단위 page-break 최소화).

## 8) 분석 로직 정의
- 기준 불량율:
  - 수량 기준: `검사불량율 = 사내불량수량 / 사내검사수량 * 100`
  - 건수 기준: `건수불량율 = (사내불량건수 + 사외부적합건수) / 출하건수 * 100`
- MA3: 최근 3개 월값 평균
- CL: 전체 월 평균
- σ: 표준편차
- UCL/LCL: `CL ± 3σ` (LCL은 0 미만 절단)
- 이상징후: 실제값이 UCL 초과 또는 LCL 미만

## 9) 수용 기준 (Acceptance Criteria)
1. 인쇄 버튼 클릭 시 고객사별 월별현황 탭만 출력된다.
2. A4 가로 기준으로 핵심 섹션이 1페이지 내 게시 가능한 밀도로 출력된다.
3. 인쇄 헤더에 고객사/기간/기준/출력일시가 표시된다.
4. 월 필터 변경 시 KPI/표/유형분포가 동일 기준으로 갱신된다.
5. 관리도에 실제값, MA3, CL, UCL/LCL이 모두 표시된다.
6. 데이터 부족 시 오류 대신 "데이터 없음" 안내가 표시된다.
